# Real-Time Code Collaboration App - Back-End

This is the back-end of a real-time code collaboration app developed using Node.js, Express.js, Mongoose, and Socket.IO. It manages JWT authentication, code synchronization, and real-time communication.

## Prerequisites

- Node.js and npm installed
- MongoDB installed and running (for data storage)

## Setup

1. Clone this repository.
2. Open a terminal and navigate to the project's root directory.
3. Run the following commands to install dependencies and start the server:

```bash
npm install
npm start
```
